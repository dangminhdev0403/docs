Response Entity

Sau chương CRUD User và ta cũng test với postman thì khi server trả về liệu dạng JSON.Tuy nhiên client nhận mỗi data dạng JSON thôi là chưa đủ

Ở chương 6 theo lí thuyết ta đã học:

  1. client gửi yêu cầu (request) lên server thông qua RestFul APIs
  2. Server(Backend) phải trả về(response) cho client(Fontent):
    • Thông tin header
    • Thông tin trạng thái(status code)
    • thông tin body(nếu có): data trả về(thường là dạng JSON)
    • Không nhất thiết tất cả các response nào đều cần đủ 3 thành phần đó

Spring ResponseEntity tài liệu tại Đây

Nếu ông nào đọc kĩ cái tài liệu mình đưa sẽ biết ngay rẳng Spring đã làm hết cho chúng ta rồi , việc của mình là chỉ có ném thông tin vào thôi :v

Status code (Mã phản hồi) tài liệu tại Đây

Phần này mọi người chỉ cần nhớ nếu Status code là con số int nếu nhỏ hơn 400 là thành công còn lơn hơn 400 là ăn lỗi.Còn chi tiết Status code như nào thì tài liệu có
