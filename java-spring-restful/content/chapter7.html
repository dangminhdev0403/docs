Phân chia dự án

Chúng ta sẽ bắt đầu với vai trò Admin , nên hãy phân chia code cho hợp lí:

  1. config: chứa những class ghi đè cấu hình mặc đình Spring
  2. controller: chứa những class xử lí request
  3. service: chứa những class xử lí logic
  4. repository: chứa những interface sẽ thao tác với database
  5. domain:chứa các đối tượng trong ứng dụng.
