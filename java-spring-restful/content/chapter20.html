CommandLineRunner

CommandLineRunner là 1 interface cho phép bạn thực thi một đoạn code cụ thể ngay sau khi ứng dụng Spring Boot khởi động xong. Interface này có một phương thức duy nhất là run(String... args), nơi bạn có thể đặt các logic cần thực thi.

Việc của ta là biến nó thành Serivce để có thể thực thi tạo bảng trong database (Nhớ if else đàng hoàng không lại tạo linh tinh).
