React chỉ là 1 thư viện ?

Ừ thì đúng rồi nó như jquery nhưng xịn xò hơn thôi.:v và React chỉ chịu trách nhiên render (vẽ UI) thông qua props/state và JSX

Vì vậy nếu ta muốn nhiều hơn (Ơ đây là từ trang này đi sang trang khác) ta cần tự làm (code js) hoặc add thêm thư viện khác (package)

Cài đặt thư viện React router Links

Mình đã có công cụ tội gì phải đi code chạy chi :v

npm install --save-exact react-router-dom@6.23.1

Lưu ý :mình đang xài phiên bản 6.23.1 thì ae vào đúng phiên bản đó mà đọc nhé, biết đâu nao vô links docs của nó update lên v7,v9 thì sao .

Tích hợp Router

    Import thứ viện vào file main.jsx

  createBrowserRouter,
  RouterProvider,

Chia lại cấu trúc dự án.

  1. Why?: vì React nó là lib như Jquery nên việc chia base dự án phụ thuộc vào chính bản thân mình.
  2. Nếu một thư viện được tích hợp sẵn việc chia cấu trúc thư mục , nó sẽ là framework(vd: nextjs ,backend:javaspring)
  3. Cách chia cấu trúc thư mục trong khoá học này chỉ mang tính chất tương đối.Điều quan trọng là bạn có khả năng mở rộng và bảo trì dự án của bạn
  4. Hãy thử làm thêm phần header , footer vào cho đủ dao diện nhé .
