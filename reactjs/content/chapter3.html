JSX(JavaScript XML) : Đây là cú pháp giúp ta code javascript trong html .

    Nguyên tắc JSX
  1. 1 component chỉ return 1 thẻ chả thôi (tức ta sẽ không có 2 hoặc nhiều thẻ html trong 1 component)
  2. Không thích ghi div vào trong thẻ đóng mở ngoặc nhọn(ví dụ: <div></div>) thì đừng ghi gì vào trong :v (Fragments) <></>
  3. thay class trong thẻ div thành className
  4. viết code javascript với { code js} nếu muốn viết trong thẻ html
