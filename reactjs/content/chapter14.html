Hướng dẫn sử dụng React Hook Form

✅ Giới thiệu

React Hook Form là thư viện giúp quản lý form trong React một cách nhẹ, dễ dùng và hiệu quả.

🔧 Cài đặt

npm install react-hook-form

📚 Cách dùng cơ bản

  • Import useForm từ thư viện
  • Dùng register để liên kết input
  • Dùng handleSubmit để xử lý submit
  • Dùng errors để hiển thị lỗi

💡 Ví dụ cơ bản

Đây là ví dụ đơn giản về form đăng ký:


import React from "react";
import { useForm } from "react-hook-form";

export default function App() {
  const { register, handleSubmit, formState: { errors } } = useForm();

  const onSubmit = data => {
    console.log(data);
  };

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input
        type="text"
        placeholder="Tên người dùng"
        {...register("username", { required: "Bắt buộc" })}
      />
      {errors.username && <p>{errors.username.message}</p>}

      <input
        type="email"
        placeholder="Email"
        {...register("email", {
          required: "Email là bắt buộc",
          pattern: { value: /^\S+@\S+$/i, message: "Email không hợp lệ" }
        })}
      />
      {errors.email && <p>{errors.email.message}</p>}

      <button type="submit">Gửi</button>
    </form>
  );
}

📎 Ghi chú

  • register(): Gắn input vào form
  • handleSubmit(): Xử lý dữ liệu gửi đi
  • formState.errors: Lỗi từng trường input
  • Form không cần dùng state riêng cho từng field
