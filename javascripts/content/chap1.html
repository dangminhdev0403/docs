SSR và CSR

Rendering

Rendering ở đây nghĩa là nói tới việc render nội dung file HTML ở server hay client.

Client Side Rendering (CSR)

Mỗi lân nhận request từ client thì server sẽ render ra 1 file html đơn giản (Với React có mỗi div root) và không có bất kì nội dung khác

  1. Ưu điểm
    • Trang web sẽ mượt mà bắt mắt hơn với Javascripts
    • Không phải load lại toàn bộ web khi chuyển trang
    • Tăng trải nhiệm người dùng
  2. Nhược điểm:
    • Không thân thiện với SEO
    • Lần tải đầu tiên sẽ phải mất nhiều thời gian do trình duyệt sử dụng JavaScript để thêm , sửa , xoá, hiện thị nội dung.

Server Side Rendering (SSR)

Ta sẽ render toàn bộ html css bên phía server trước rồi mới gửi xuống client

  1. Ưu điểm
    • Thời gian tải nhanh
    • Tối ưu SEO
    • Tăng trải nhiệm người dùng
  2. Nhược điểm:
    • Mất thời gian khi chuyển sang 1 trang khác
