Cấu hình file launch.json cho dự án Reactjs trong VS Code

runtimeExecutable : sẽ là đường dẫn đến browser.exe trong pc anh em

Mình xài cốc cốc và đường dẫn đến file browser.exe trong pc là:
C:\Program Files\CocCoc\Browser\Application\browser.exe.

hãy thêm / vào nhé 

{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Debug Vite + React with Coc Coc",
      "type": "chrome",
      "request": "launch",
      "url": "http://localhost:5173",
      "webRoot": "${workspaceFolder}/src",
      "runtimeExecutable": "C:\\Program Files\\CocCoc\\Browser\\Application\\browser.exe",
      "breakOnLoad": true,
      "sourceMaps": true,
      "trace": true,
      "skipFiles": ["<node_internals>/**"]
    }
  ]
}

Cấu hình xong ấn F5 để vào debug
